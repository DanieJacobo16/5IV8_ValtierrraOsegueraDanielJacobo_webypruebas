Todos los formularios en la web, manejan metodos para el envio de las diferentes peticiones que realiza el usuario condirendado la arquitrectura cliente servidor se hacen por el metodo http y acualmente https, cada peticion debe ser atendida por un tiempo de respuesta conforme al modelo TCP/LP Los metodos principales de los formularoios son :

    • GEt

      Este metodo se encarga de enviar informacion del formulario por medio de la salida estandar del navgegador, esto propicia que pueda existir inyecciones de codigo realizando violaciones a la seguridad al sistema

      • POST

      Este metodo se encarga de enviar la informacion para para poder registrar un cambio o un nuevo registro dentro del sistema enviandolo igual por el metodo estandar, por medio del ocultamientom esto significa que no es visible, para el usuario lo que nos ofrece mayor grado de seguridad en el manjo de la informacion


    En resumen los puntos son:

    • Get: Sirve para enviar o realizar operaciones de consulta
      Post: Sirve para enviar un recurso al servidor con el fin de registrarlo
      Delete: Sirve para eliminar un recurso especifico dentro del servidor obteniendo una respuesta de actualizacion
    Put: Sirve para almacenar el recurso de una informacion particular y conectarse al servicio para una atualizacion
